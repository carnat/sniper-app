## Level System Overview
The Level System is the Commander's wealth milestone framework.
Each Level has a liquid portfolio target measured in Thai Baht (฿).
"Liquid" = Arsenal stock value + Vault E-class × 0.80.
Locked Vault assets (SSF/RMF/SSFA/SSFE) do NOT count toward Level milestones.
